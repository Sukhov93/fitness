# SoftLine — демо-лендинг студии женского фитнеса

Готовая статика: `index.html` + папка `media` с видео. Ничего собирать не нужно, открывается прямо в браузере.

## Как выложить на GitHub Pages (2 минуты, без консоли)

1. Зайди на github.com → **New repository** (например, назови `fitness-landing`)
2. Выбери **Public**, не добавляй README/gitignore — оставь пустым
3. Открой созданный репозиторий → **Add file → Upload files**
4. Перетащи туда `index.html` и папку `media` целиком (важно сохранить структуру: `media/testimonial-1.mp4`, `media/workout-1.mp4` — пути в HTML прописаны относительно корня)
5. Нажми **Commit changes**
6. Перейди в **Settings → Pages**
7. В разделе **Branch** выбери `main` и папку `/ (root)` → **Save**
8. Через 1–2 минуты сайт появится по адресу:
   `https://<твой-логин>.github.io/fitness-landing/`

Ссылка появится прямо в разделе Pages после сохранения — просто обнови страницу.

## Если захочешь обновить видео или тексты позже

- Видео лежат в `media/`, просто замени файлы с теми же именами
- Весь текст, цвета и структура блоков — в `index.html`, редактируется как обычный текстовый файл прямо в GitHub (кнопка карандаша на странице файла)
